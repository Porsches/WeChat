# WeChat帮助插件


## 站点安装


``` bash

$ git clone https://github.com/xulayen/WeChat.git & cd Wechat


$ npm install


$ npm start


```

## 站点执行


浏览器中打开 http://localhost:3000/index.html


## npm下载

``` bash

$ npm install jquery_wechat_sdk


```

## 使用

``` bash

var wx = require('jquery_wechat_sdk');

```
